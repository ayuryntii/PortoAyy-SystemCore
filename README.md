> **"A futuristic, HUD-inspired personal portfolio engineered for the modern developer."**

Selamat datang di *repository* proyek portofolio saya! Website ini merupakan representasi digital interaktif yang dieksekusi dan dibangun secara *native* menggunakan *framework* **Laravel**. 

Mengusung visual *System Core / Cyber HUD* yang kental, sistem ini tidak hanya sebatas *Curriculum Vitae* konvensional, melainkan wujud aktualisasi teknis (showcase) dari pemahaman mendalam pada aspek manipulasi *Front-end logic*, *Back-end scripting*, serta seni arsitektur UI/UX bernuansa futuristik.

---

## ✨ System Features (Modul Utama)

- 🌐 **Cinematic HUD Interface:** Menghadirkan antarmuka pengguna ala layar kendali *sci-fi*, dinamis dengan animasi lekuk *scanline*, dan *glassmorphism* tingkat lanjut.
- 👤 **Data Terminal (About & Education):** Visualisasi data rekaman akademik serta peta kompetensi (skill) IT yang terstruktur dengan elegan, bersih, dan memanjakan mata.
- 📂 **Project Vault (Portfolio):** Etalase eksklusif untuk mendemonstrasikan proyek lintas teknis (e.g. Sistem Presensi otomatis, *Engine* Rekomendasi Karir, hingga Manajemen Akademik).
- 🖨️ **Automated CV Generator Engine:** Terintegrasi dengan injeksi pemroses dokumen cerdas untuk menyusun profil *Curriculum Vitae* berformat *PDF document* secara *real-time* via *1-click extraction*.

## 🛠️ Booting Sequence (Local Evaluation)

Apabila Anda tertarik untuk menguji performansi "System Core" ini secara lokal tanpa kendala, pastikan *environment* ekosistem (*machine*) Anda telah dilengkapi basis `PHP` dan `Composer`. Selanjutnya, cukup jalankan siklus *booting* di bawah ini:

**1. Install Core Dependencies**
Buka layanan terminal pada direktori bawaan (*root*) proyek, lalu eksekusi arahan *package manager*:
> `composer install`

**2. Environment Configuration**
Lakukan duplikasi variabel identitas lingkungan (*environment variables*) agar memori sistem terhubung stabil:
> `cp .env.example .env`

**3. Inject Security Key**
Bangkitkan lapisan sertifikasi keamanan lokal aplikasi Anda:
> `php artisan key:generate`

**4. Activate Core Server**
Inisialisasi aliran virtual server agar portal sistem bisa diakses (*online*):
> `php artisan serve`

> *Sistem operasional kini dalam mode siaga! Akses visual terminal melalui peramban (browser) di koordinat portal:* `http://localhost:8000`
