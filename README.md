> **"A futuristic, HUD-inspired personal portfolio designed for the modern developer."**

Selamat datang di repository proyek portofolio saya. Website ini adalah representasi digital interaktif yang dibangun secara *native* di atas framework **Laravel**. 

Mengusung visual *System Core / Cyber HUD*, proyek ini tidak hanya berfungsi sebagai *Curriculum Vitae* digital, tetapi juga sebagai bukti nyata (showcase) mengenai pemahaman mendalam terkait desain UI/UX, *Front-end logic*, dan *Back-end scripting*.

---

## ✨ Fitur Utama (Core Modules)

- 🌐 **Cinematic HUD Interface:** Menghadirkan antarmuka pengguna bergaya futuristik dengan efek *scanline* dan *glassmorphism* tingkat lanjut.
- 👤 **Data Terminal (About & Education):** Visualisasi data rekaman akademik dan *skill* IT yang intuitif dan profesional.
- 📂 **Project Vault (Portfolio):** Etalase *high-fidelity* untuk menampilkan proyek teknis seperti Sistem Presensi QR, Engine AI, dan Web Manajemen.
- 🖨️ **PDF CV Generator Engine:** Terintegrasi dengan pemroses dokumen otomatis untuk menghasilkan *Curriculum Vitae* format PDF secara *real-time* dan *1-click download*.
